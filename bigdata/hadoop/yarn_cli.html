$ Yarn Cli命令

$ list:按状态过滤

yarn application -list -appStates ALL

合法的application 状态:

  • ALL
  • NEW
  • NEW_SAVING
  • SUBMITTED
  • ACCEPTED
  • RUNNING
  • FINISHED
  • FAILED
  • KILLED

输出示例：
