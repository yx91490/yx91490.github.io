# JavaMailдҪҝз”ЁжҖ»з»“

еңЁе·ҘдҪңдёӯдҪҝз”ЁjavaеҸ‘йҖҒиҝҮйӮ®д»¶зҡ„дәәдёҖиҲ¬йғҪзҹҘйҒ“javamailиҝҷдёӘдёңиҘҝ,д»–жҳҜJavaEEжҸҗдҫӣзҡ„дёҖеҘ—жһ„е»әйӮ®д»¶е’Ңж¶ҲжҒҜеә”з”Ёзҡ„API,дҪҶжҳҜз”ЁиҝҮзҡ„дәәйғҪзҹҘйҒ“иҝҷеҘ—APIжҳҜеӨҡд№Ҳзҡ„йҡҫз”Ё,еҲҡејҖе§Ӣз”Ёзҡ„ж—¶еҖҷжҲ‘йғҪжҖҖз–‘е…Ёдё–з•Ңзҡ„зЁӢеәҸе‘ҳзҡ„йғҪжңүиҝҷд№Ҳејәзҡ„еҝҚиҖҗеҠӣеҗ—?дёәд»Җд№ҲжІЎжңүдәәйҮҚж–°еҸ‘жҳҺдёӘиҪ®еӯҗ?йҡҫйҒ“е°ұжҳҜеӣ дёәд»–жҳҜOracleе®ҳж–№еҮәе“Ғзҡ„дёңиҘҝеҗ—:(?

дҪҶжҳҜи¶ҠжҳҜжҒ¶еҝғзҡ„дёңиҘҝе°ұи¶Ҡжғід»ҘеҗҺйҒҝе…ҚеҶҚиё©еҲ°е®ғ.дәҺжҳҜжңүдәҶе°ҒиЈ…дёҖдёӢиҝҷдёӘдёңиҘҝзҡ„жғіжі•(дёҚиҝҮдәӢеҗҺиҜҒжҳҺиҝҷдёӘдёңиҘҝж— и®әдҪ жҖҺд№Ҳе°ҒиЈ…зңӢиө·жқҘиҝҳжҳҜйӮЈд№Ҳдё‘йҷӢ...),йңҖжұӮеҳӣеҫҲз®ҖеҚ•,иғҪеӨҹеҸ‘йҖҒйӮ®д»¶е°ұиЎҢ,йӮ®д»¶еҶ…е®№ж”ҜжҢҒеҶ…еөҢзҡ„еӣҫзүҮ,иҝҳеҸҜд»ҘеёҰдёҠеҮ дёӘйҷ„д»¶е°ұз®—е®ҢзҫҺдәҶ.дәӢе®һдёҠ,жңүдёҖе ҶRFCж–ҮжЎЈжқҘжҸҸиҝ°emailзҡ„е…·дҪ“ж јејҸ,дҪҶжҳҜжҲ‘д»¬еҸҜиғҪзңҹжӯЈз”ЁдёҚдёҠйӮЈд№Ҳе…ЁйқўиҜҰз»Ҷзҡ„еҠҹиғҪ.

иӮҜе®ҡжІЎжңүеҝғжҖқзңӢеҶ—й•ҝзҡ„иӢұж–ҮRFCж–ҮжЎЈ,зҪ‘дёҠжүҫжқҘдёҖзҜҮзңӢиө·жқҘдёҚй”ҷзҡ„ж–ҮжЎЈ (opens new window),иҰҒжһ„е»әеӨҡеұӮзҡ„е®№еҷЁжқҘжҠҠдёҚеҗҢзұ»еһӢзҡ„иө„жәҗеҢ…иЈ…иө·жқҘ,жҜ”еҰӮдҪ иҰҒеҸ‘дёҖе°ҒеёҰеӣҫзүҮд№ҹеёҰйҷ„д»¶зҡ„йӮ®д»¶,дҪ иҰҒиҝҷд№ҲеҢ…иЈ…:
