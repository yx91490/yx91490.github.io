# JNI学习笔记

## 介绍

使用 java native 方法的场景：

- 支持依赖平台的特性
- 复用另一种语言的库
- 用一种底层语言实现高效率的代码

JNI 可以实现：

- 创建，检查，更新java 对象（包括数组，string）
- 调用Java 方法
- 捕获和抛出异常
- 应用运行时类型检查
